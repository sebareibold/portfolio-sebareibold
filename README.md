# Portafolio Personal – Sebastián Alejandro Reibold

Este es mi portafolio personal desarrollado con **Next.js**, donde muestro quién soy, mis habilidades como desarrollador y algunos de los proyectos que he realizado. Es un espacio pensado para compartir mi trabajo y mi crecimiento profesional en el mundo del desarrollo de software.

## 🧑‍💻 Sobre mí

Soy **Sebastián Alejandro Reibold**, estudiante de Licenciatura en Ciencias de la Computación en la Universidad Nacional del Comahue y desarrollador frontend con conocimientos en backend. Me apasiona crear experiencias únicas y funcionales en la web.

## 🚀 Tecnologías y Herramientas

- **Frontend:** React.js, Next.js, HTML, CSS, Tailwind CSS
- **Backend:** Node.js, Express (básico), Firebase
- **Otros:** Git, GitHub, Figma, Vite, React Router

## 📁 Estructura del Portafolio

El portafolio incluye:

- Una sección **sobre mí** con una pequeña descripción personal.
- Mis **habilidades técnicas**.
- Una galería de **proyectos destacados**.
- Información de **contacto**.
